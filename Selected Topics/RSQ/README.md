# Range Sum Query

### Segment Tree Time/Space Complexity

```
| -------- | ------------ |
| Build    | O(n)         |
| Query    | O(log n)     |
| Update   | O(log n)     |
| Space    | O(n) (≈ 4n)  |
| -------- | ------------ |
```

### SQRT Decomposition
```
| -------- | ------------ |
| Build    | O(n)         |
| Update   | O(1)         |
| Query    | O(√n)        |
| Space    | O(n)         |
| -------- | ------------ |
```

### Binary Indexed Tree 

```
| -------- | ------------------- |
| Build    | O(n) sau O(n log n) |
| Update   | O(log n)            |
| Query    | O(log n)            |
| Space    | O(n)                |
| -------- | ------------------- |
```
